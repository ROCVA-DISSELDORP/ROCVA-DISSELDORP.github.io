Unity VR Roller Coaster Shooter Tutorial

Part 2: Targets, Scoring, and Game Loop

← Back to Part 1
1. Create the Shooting Target Prefab

1. Build the Target Model

  • Create an empty GameObject and name it 'ShootingTarget-Prefab'.
  • Create a 'Cylinder' 3D object, name it 'Target', and parent it to 'ShootingTarget-Prefab'.
  • Set the 'Target' transform:
    • Position: (0, 2, 0)
    • Rotation: (90, 0, 0)
    • Scale: (1, 0.05, 1)
  • On the 'Target', remove the 'Capsule Collider' and add a 'Mesh Collider', then check the 'Convex' box.

2. Create and Apply Material

  • Create a new Material named 'ShootingTarget-Mat'.
  • Set its 'Albedo' color to bright red.
  • Enable 'Emission', set the color to red, and the 'Intensity' to 1.
  • Apply this material to the 'Target' GameObject.

3. Add Effects

  • Create a 'Particle System' and an 'Audio Source'.
  • Parent both to the 'ShootingTarget-Prefab' and set their position to (0, 2, 0).
2. Setup Target Behavior and Scoring

1. Configure Target Effects

Select the 'Particle System' on the target and apply these main settings:


Duration: 1.0
Looping: false
Play on Awake: false
Start Lifetime: 2.0
Start Speed: 8.0
Start Size: 0.2
Gravity Modifier: 1.0
// Emission Module
Rate over Time: 0
Bursts: Add one burst of 30 particles
// Shape Module
Shape: Sphere
Radius: 0.5
// Size over Lifetime Module (Enabled)
Curve: Linear from 1 down to 0
// Collision Module (Enabled)
Type: World

For the 'Audio Source', assign a sound to the 'AudioClip' field (e.g., 'Button Pop.wav') and uncheck 'Play On Awake'.

2. Create Target.cs Script

Create a new C# script named 'Target.cs'. This script handles what happens when the target is hit, including playing effects and telling the GameManager to increase the score.


using System.Collections;
using System.Collections.Generic;
using UnityEngine;
using UnityEngine.Events;

public class Target : MonoBehaviour
{
    public int targetPointsValue = 100;
    public ParticleSystem targetParticleSystem;
    public AudioSource targetAudioSource;
    public UnityEvent onTargetHitEvent;

    private GameManager gameManager;

    void Start()
    {
        // Find the GameManager in the scene when the game starts
        gameManager = FindObjectOfType<GameManager>();
    }
    
    // Called automatically by Unity's physics system on collision
    private void OnCollisionEnter(Collision collider)
    {
        Debug.Log("Target was hit by: " + collider.gameObject.name);

        // Play hit effects
        targetParticleSystem.Play();
        targetAudioSource.Play();

        // Add points to the player's score via the GameManager
        if (gameManager != null)
        {
            gameManager.IncreasePlayerScore(targetPointsValue);
        }

        // Invoke any events assigned in the Inspector (like starting the ride)
        if (onTargetHitEvent != null)
        {
            onTargetHitEvent.Invoke();
        }
        
        // Disable the target so it can't be hit again
        GetComponent<MeshCollider>().enabled = false;
        GetComponent<MeshRenderer>().enabled = false;
    }
}

3. Finalize the Prefab

  • Add the 'Target.cs' script to the 'Target' child GameObject.
  • Drag the 'Particle System' and 'Audio Source' into the corresponding fields on the 'Target' script in the Inspector.
  • Drag the parent 'ShootingTarget-Prefab' from the Hierarchy into your Prefabs folder to save it. You can now delete it from the scene.
3. Create the GameManager and Game Loop

1. Create the GameManager Object and Script

Create an empty GameObject named 'GameManager'. Then, create a C# script named 'GameManager.cs' and add it to the object. This script will control the overall game state, including starting/stopping the ride, managing the UI, and handling scene changes.


using System.Collections;
using System.Collections.Generic;
using UnityEngine;
using UnityEngine.Splines;
using UnityEngine.SceneManagement;

public class GameManager : MonoBehaviour
{
    // Assign these in the Unity Inspector
    public VRPlayer player;
    public RollerCoasterTrain rollerCoasterTrain;
    public GameObject tutorialCanvas;
    public GameOverCanvas gameOverCanvas;
    public PlayerScoreCanvas playerScoreCanvas;

    void Start()
    {
        // Hide the game over screen at the start
        gameOverCanvas.gameObject.SetActive(false);
    }

    public void StartRollerCoasterRide()
    {
        rollerCoasterTrain.GetComponent<SplineAnimate>().Play();
        tutorialCanvas.SetActive(false); // Hide the tutorial text
    }

    public void StopRollerCoasterRide()
    {
        rollerCoasterTrain.GetComponent<SplineAnimate>().Pause();
    }

    public void GameOver()
    {
        gameOverCanvas.gameObject.SetActive(true); // Show the game over screen
    }

    public void IncreasePlayerScore(int amount)
    {
        player.playerScore += amount;
        // Update the score display
        playerScoreCanvas.playerScoreValueText.text = player.playerScore.ToString();
    }

    // --- Scene Management ---
    public void LoadScene(string sceneName)
    {
        SceneManager.LoadScene(sceneName);
    }

    public void LoadNextScene()
    {
        int nextSceneIndex = SceneManager.GetActiveScene().buildIndex + 1;
        // If there is a next scene, load it. Otherwise, loop back to the main menu (index 0).
        if (SceneManager.sceneCountInBuildSettings > nextSceneIndex)
        {
            SceneManager.LoadScene(nextSceneIndex);
        }
        else
        {
            SceneManager.LoadScene(0);
        }
    }

    public void RestartScene()
    {
        SceneManager.LoadScene(SceneManager.GetActiveScene().name);
    }
}

You will also need a simple script for the player to hold the score. Create 'VRPlayer.cs' and add it to your 'VRPlayer' GameObject.


using UnityEngine;

public class VRPlayer : MonoBehaviour
{
    public int playerScore = 0;
}

2. Place Targets and Triggers in the Scene

  • Create an empty GameObject named 'ShootingTargets' to keep your hierarchy clean.
  • Drag instances of your 'ShootingTarget-Prefab' from the Prefabs folder into the scene and parent them to 'ShootingTargets'.
  • Place one target directly in front of the player's starting position. This will be the "start" button.
  • Place about 50 other targets around your roller coaster track.

3. Hook up the Start Event

Select the first shooting target you placed. In the Inspector, find the 'On Target Hit Event' on its 'Target' script component.

  • Click the '+' to add an event.
  • Drag the 'GameManager' GameObject into the object field.
  • From the function dropdown, select GameManager > StartRollerCoasterRide().
  • Important: On your 'RollerCoasterTrain', make sure the 'Spline Animate' component has 'Play On Awake' unchecked.
4. Create a Reusable Event Trigger

1. Create the Trigger Object and Script

This trigger will be an invisible box that can fire events when the player enters it. We'll use it to end the ride.

  • Create a 'Cube' 3D object and name it 'CustomEventTrigger-Prefab'.
  • In the Inspector, uncheck the 'Mesh Renderer' to make it invisible.
  • On the 'Box Collider', check the 'Is Trigger' box.
  • Create a C# script named 'CustomEventTrigger.cs' and add it to the object.

using UnityEngine;
using UnityEngine.Events;

public class CustomEventTrigger : MonoBehaviour
{
    // The tag of the object that will activate this trigger
    public string triggerTag = "Player";

    // Events that can be assigned in the Inspector
    public UnityEvent onTriggerEnterEvent;
    
    // Draws a visible box in the Scene view so we can see the trigger area
    void OnDrawGizmos()
    {
        Gizmos.color = new Color(0, 1, 1, 0.5f); // Cyan
        Gizmos.matrix = transform.localToWorldMatrix;
        Gizmos.DrawCube(Vector3.zero, Vector3.one);
    }
    
    // Called when another collider enters this trigger
    private void OnTriggerEnter(Collider other)
    {
        // Check if the entering object has the correct tag
        if (other.CompareTag(triggerTag) && onTriggerEnterEvent != null)
        {
            Debug.Log(other.name + " entered the trigger.");
            onTriggerEnterEvent.Invoke();
        }
    }
}

2. End the Ride with the Trigger

  • Tag your 'VRPlayer' GameObject with the 'Player' tag. If it doesn't exist, create it via 'Add Tag...'.
  • Place an instance of the 'CustomEventTrigger-Prefab' near the end of your roller coaster track. Scale it so the player will definitely pass through it.
  • Select the trigger in the Hierarchy. Find the 'On Trigger Enter Event' on its script component.
  • Click '+' and drag the 'GameManager' into the object field.
  • From the function dropdown, select GameManager > StopRollerCoasterRide().
  • Add another event by clicking '+' again. Drag in the 'GameManager' and select GameManager > GameOver().
5. Create the Game UI

1. Create the Tutorial Canvas

This canvas will show instructions at the start of the game.

  • Create a new UI Canvas (GameObject > UI > Canvas) and name it 'TutorialCanvas'.
  • In the Inspector, set its 'Render Mode' to 'World Space'.
  • Position it in front of the player's starting view. A good starting RectTransform is:
    • Position: (0, 2.3, 5), Scale: (0.001, 0.001, 0.001)
  • Add a 'Panel' and 'Text - TextMeshPro' object to the canvas.
  • Write your tutorial text, e.g., "Grab the gun and shoot the target to start!".

2. Create the Game Over Canvas

  • Create another 'World Space' canvas named 'GameOverCanvas'. Position it near the end of the ride.
  • Add a dark, semi-transparent 'Panel' to serve as a background.
  • Add 'Text - TextMeshPro' elements for "GAME OVER!", "YOUR SCORE:", and a placeholder for the score value (e.g., "99999").
  • Add three UI Buttons: 'Button-Quit', 'Button-Restart', and 'Button-NextRide'.
  • Crucially, add the 'Tracked Device Graphic Raycaster' component to this canvas so you can interact with the buttons in VR.
  • Create an empty script 'GameOverCanvas.cs' and add it to the canvas object. (It can remain empty for now).

3. Create the Player Score Canvas

  • Create a small 'World Space' canvas named 'PlayerScoreCanvas'.
  • Parent it to the 'RollerCoasterTrain' so it moves with the player.
  • Position it in the player's view, like a dashboard display. A good starting RectTransform is:
    • Position: (0, 1.6, 0.575), Scale: (0.001, 0.001, 0.001), Width: 200, Height: 50
  • Add a 'Panel' and a 'Text - TextMeshPro' object to display the score, starting with "0".
  • Create 'PlayerScoreCanvas.cs', add it to the canvas, and add this code to it:

using UnityEngine;
using TMPro;

public class PlayerScoreCanvas : MonoBehaviour
{
    // Assign the score Text object to this field in the Inspector
    public TMP_Text playerScoreValueText;
}
6. Create the Main Menu & Build the Game

1. Create a New Scene for the Main Menu

  • Create a new scene and save it as 'MainMenu.unity'.
  • Add a simple 'Cube' to act as a floor.
  • Add your 'VRPlayer' prefab to the scene.
  • Create a 'World Space' canvas similar to the Game Over screen, but with "Play", "Settings", and "Quit" buttons. Add the 'Tracked Device Graphic Raycaster' component.

2. Script the Main Menu Buttons

Create a C# script named 'MainMenuCanvas.cs', add it to your main menu canvas, and paste this code:


using UnityEngine;
using UnityEngine.SceneManagement;

public class MainMenuCanvas : MonoBehaviour
{
    // Loads the scene with the given name (e.g., "Level 1")
    public void LoadScene(string sceneName)
    {
        SceneManager.LoadScene(sceneName);
    }

    // Quits the application
    public void QuitGame()
    {
        Application.Quit();
        #if UNITY_EDITOR
        UnityEditor.EditorApplication.isPlaying = false; 
        #endif
    }
}

Hook up the buttons in the Inspector:

  • Play Button: OnClick() -> MainMenuCanvas.LoadScene("Level 1")
  • Quit Button: OnClick() -> MainMenuCanvas.QuitGame()

3. Build and Run the Game

  • Go to File > Build Settings.
  • Click 'Add Open Scenes' to add your Main Menu. Then open your 'Level 1' scene and add it as well.
  • Drag 'MainMenu' to the top of the list (index 0) so it loads first.
  • Connect your Quest headset via USB.
  • Under 'Run Device', select your headset.
  • Click 'Build and Run' to compile the game and install it on your headset.

VRGun.cs


using System.Collections;
using System.Collections.Generic;
using UnityEngine;

public class VRGun: MonoBehaviour
{
    public Transform gunSnapTransform;
    public Transform bulletSpawnTransform;
    public GameObject bulletPrefab;
    public ParticleSystem gunFireParticleSystem;
    public AudioSource gunFireAudioSource;
    public Collider[] gunColliders;
    
    public float fireSpeed = 125.0f;

    private Vector3 gunVelocity;
    private Vector3 previousPosition;
    private bool fireFlag = false;

    void Start()
    {
        previousPosition = bulletSpawnTransform.position;
    }

    void Update()
    { 
        gunVelocity = (bulletSpawnTransform.position - previousPosition) / Time.deltaTime;
        previousPosition = bulletSpawnTransform.position;
    }

    void FixedUpdate()
    {
        if(fireFlag == true)
        {
            fireFlag = false;
            Fire();
        }
    }

    public void FireBullet()
    {
        fireFlag = true;
    }

    public void Fire()
    {
        gunFireParticleSystem.Play();
        gunFireAudioSource.Play();

        GameObject spawnedBullet = Instantiate(bulletPrefab);

        spawnedBullet.transform.position = bulletSpawnTransform.position;
        spawnedBullet.transform.rotation = bulletSpawnTransform.rotation;

        spawnedBullet.GetComponent<Rigidbody>().velocity = gunVelocity + (spawnedBullet.transform.forward * fireSpeed);
        
        Destroy(spawnedBullet, 5);

        SphereCollider spawnedBulletCollider = spawnedBullet.GetComponent<SphereCollider>();

        for(int i=0; i<gunColliders.Length; i++)
        {
            Physics.IgnoreCollision(spawnedBulletCollider, gunColliders[i], true);
        }
    }

    public void OnGrab()
    {
        Debug.Log("Gun is Grabbed!!!");
    }

    public void OnRelease()
    {
        Debug.Log("Gun is Dropped!!!");
        transform.position = gunSnapTransform.position;
        transform.rotation = gunSnapTransform.rotation;
        GetComponent<Rigidbody>().isKinematic = true;
    }
}

CustomEventTrigger.cs


using System.Collections;
using System.Collections.Generic;
using UnityEngine;
using UnityEngine.Events;

public class CustomEventTrigger : MonoBehaviour
{
    public string triggerTag = "Player";
    public UnityEvent onTriggerEnterEvent;
    public UnityEvent onTriggerStayEvent;
    public UnityEvent onTriggerExitEvent;

    void OnDrawGizmos()
    {
        Gizmos.color = new Color(0, 1, 1, 0.5f);
        Gizmos.matrix = transform.localToWorldMatrix;
        Gizmos.DrawWireCube(Vector3.zero, Vector3.one);
    }

    IEnumerator OnTriggerEnter(Collider collider)
    {
        if (onTriggerEnterEvent != null && collider.gameObject.tag == triggerTag)
        {
            onTriggerEnterEvent.Invoke();
        }
        yield return null;
    }

    IEnumerator OnTriggerStay(Collider collider)
    {
        if (onTriggerStayEvent != null && collider.gameObject.tag == triggerTag)
        {
            onTriggerStayEvent.Invoke();
        }
        yield return null;
    }

    IEnumerator OnTriggerExit(Collider collider)
    {
        if (onTriggerExitEvent != null && collider.gameObject.tag == triggerTag)
        {
            onTriggerExitEvent.Invoke();
        }
        yield return null;
    }
}
