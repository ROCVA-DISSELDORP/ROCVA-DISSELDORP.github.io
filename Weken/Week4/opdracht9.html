Opdracht 9: Documenteer je JavaScript

Beschrijving

In deze opdracht ga je je JavaScript-code voorzien van comments en documentatie. Dit helpt bij het begrijpen van de logica en structuur van je script. Gebruik W3Schools om javascript te begrijpen.

Stappen:

  1. Open je JavaScript-bestand. Als je nog geen bestand hebt, kun je de volgende voorbeeldcode gebruiken: 
    function greet() {
    console.log("Hello, world!");
}

greet();
  2. Voeg comments toe aan je JavaScript-code om uit te leggen wat elke functie en regel doet. Gebruik de volgende syntax: 
    // Dit is een comment in JavaScript
    Bijvoorbeeld: 
    // Een functie die een begroeting logt naar de console
function greet() {
    console.log("Hello, world!"); // Print "Hello, world!" naar de console
}

// Roep de functie aan
greet();
  3. Controleer of je comments logisch en nuttig zijn.

Resultaat:

Je JavaScript-bestand bevat duidelijke comments en is goed gedocumenteerd.
