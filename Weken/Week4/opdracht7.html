Opdracht 7: Documenteer je HTML

Beschrijving

In deze opdracht ga je je HTML-bestand voorzien van comments en documentatie. Dit maakt je code overzichtelijker en makkelijker te begrijpen. Gebruik w3schools om alles op te zoeken over html W3Schools

Stappen:

  1. Open je HTML-bestand. Html bestaat uit nested elements. Elk element heeft een tag en een set van attributen. In het element zit tekst of andere geneste elementen. Alle elementen samen noem je we de DOM (Document Object Model). 
    <element1 attribute1="waarde1" attribute2="waarde2">
    Tekst1
    <element2 attribute1="waarde1" attribute2="waarde2">Tekst2</element2>
</element1>
  2. Voeg comments toe aan je HTML-code om uit te leggen wat de elementen en attributen doen. Gebruik de volgende syntax: 
    <!-- Dit is een comment in HTML -->
    Bijvoorbeeld: 
    <!-- Titel van de pagina --><title>Voorbeeldpagina</title>
    <title>Voorbeeldpagina</title>
  3. Controleer of je comments nuttig en duidelijk zijn.

Resultaat:

Je HTML-bestand bevat duidelijke comments en is overzichtelijker geworden.
