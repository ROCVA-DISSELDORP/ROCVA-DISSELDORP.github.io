Opdracht 8: Documenteer je CSS

Beschrijving

In deze opdracht ga je je CSS-bestand voorzien van comments en documentatie. Dit helpt om stijlen te begrijpen en aan te passen. Gebruik W3Schools om css te begrijpen.

Stappen:

  1. Open je CSS-bestand. Als je nog geen bestand hebt, kun je de volgende voorbeeldcode gebruiken: 
    body {
    background-color: blue;
}

h1 {
    color: white;
}
  2. Voeg comments toe aan je CSS-code om uit te leggen wat de stijlen doen. Gebruik de volgende syntax: 
    /* Dit is een comment in CSS */
    Bijvoorbeeld: 
    /* De achtergrondkleur van de pagina is blauw */
body {
    background-color: blue;
}

/* De kleur van de titeltekst is wit */
h1 {
    color: white;
}
  3. Controleer of je comments duidelijk en begrijpelijk zijn.

Resultaat:

Je CSS-bestand bevat duidelijke comments en is beter gedocumenteerd.
