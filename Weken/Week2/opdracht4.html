Opdracht 4: Eerste Prototype Online op GitHub Pages

Beschrijving

In deze opdracht maak je een repository op GitHub en zet je jouw eerste prototype online met GitHub Pages.

Stappen:

  1. Log in op GitHub en klik op "New Repository".
  2. Geef je repository de naam your-username.github.io en kies de optie Public. Klik vervolgens op "Create Repository".
  3. Upload je bestanden:
    • Open de repository en klik op "Add file" → "Upload files".
    • Upload de HTML-, CSS- en JavaScript-bestanden van je prototype. Of begin simpel met 
      <h1>Hello World</h1>
    • Klik op "Commit changes" om ze op te slaan.
  4. Ga naar het tabblad "Action" van je repository en je eerste website is nu aan het deployen.
  5. Als alle vinkjes groen zijn ga dan naar your-username.github.io.
  6. Clone de repository naar je computer door de link te kopiëren die te vinden is door op de groene knop te klikken.
  7. In vs code open je de map waar je de repository wilt downloaden
  8. Open een terminal en run git clone https://github.com/your-username/your-username.github.io
  9. Maak een aantal aanpassingen in je HTML-bestanden en upload ze met git add . en git commit -m "Nieuwe aanpassingen"
  10. Upload de wijzigingen met git push

Resultaat:

Je eerste prototype staat online! en is toegankelijk via een GitHub Pages-link. En je kan lokaal aanpassingen maken en uploaden via git.
