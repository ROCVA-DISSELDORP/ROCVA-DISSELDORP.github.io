Opdracht 3: GitHub Account Maken & Git Installeren

Beschrijving

In deze opdracht maak je een GitHub-account aan en installeer je Git om later met je projecten te werken.

Stappen:

  1. Ga naar de GitHub-website en klik op "Sign Up" om een account aan te maken.
  2. Download en installeer Git van de officiële Git-website.
  3. Open een terminal (bijvoorbeeld PowerShell of Git Bash) en controleer of Git is geïnstalleerd met het commando: 
    git --version
  4. Stel je Git-configuratie in door je naam en e-mailadres in te voeren (gebruik hetzelfde e-mailadres als je GitHub-account): 
    
git config --global user.name "Jouw Naam"
git config --global user.email "jouw@email.com"
  5. Log in bij GitHub via de terminal met: 
    git login

Resultaat:

Je hebt een werkend GitHub-account en Git geïnstalleerd en geconfigureerd op je computer.
