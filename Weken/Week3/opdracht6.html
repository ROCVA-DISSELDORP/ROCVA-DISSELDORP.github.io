Opdracht 6: Basisinstellingen in VS Code

Beschrijving

In deze opdracht pas je de basisinstellingen van Visual Studio Code aan om je werkruimte persoonlijker en efficiÃ«nter te maken.

Stappen:

  1. Thema en Lettertype:
    • Open de instellingen via Ctrl + , of het menu File > Preferences > Settings.
    • Zoek naar "Color Theme" en kies een thema dat je prettig vindt, zoals "Dark+" of "Light+".
    • Pas de lettergrootte aan (bijvoorbeeld 14px of 16px).
    • Optioneel: pas je lettertype aan (bijvoorbeeld "Fira Code" of "JetBrains Mono").
  2. Extensies Installeren:
    • Open de extensiemarkt met Ctrl + Shift + X.
    • Installeer de extensies:
      • Prettier: Automatische codeformattering. (Shift + Alt + F om een document nu te formatteren)
      • Live share: Open je browser en ga naar https://vscode.dev/share om te delen.
    • Activeer of deactiveer een extensie en kijk wat er verandert in je editor.
  3. Auto Save:
    • Schakel Auto Save in via File > Auto Save of zoek in de instellingen naar "Auto Save".
    • Selecteer de optie After Delay.
  4. Optioneel: Snippet Maken:
    • Open de command palette met Ctrl + Shift + P.
    • Typ en selecteer Preferences: Configure User Snippets.
    • Kies een taal waarin je de snippet wilt toevoegen, bijvoorbeeld HTML.
    • Er opent een JSON-bestand. Voeg het volgende toe binnen de bestaande haakjes: 
      
    "HTML5 Skeleton": {
        "prefix": "html5",
        "body": [
            "<!DOCTYPE html>",
            "<html lang=\\"en\\">",
            "<head>",
            "    <meta charset=\\"UTF-8\\">",
            "    <meta name=\\"viewport\\" content=\\"width=device-width, initial-scale=1.0\\">",
            "    <title>$1</title>",
            "</head>",
            "<body>",
            "    $0",
            "</body>",
            "</html>"
        ],
        "description": "HTML5 basisdocument"
    }
    • Sla het bestand op.
    • Test de snippet door in een nieuw HTML-bestand html5 te typen en de snippet te selecteren.

Resultaat:

Je hebt de basisinstellingen aangepast en je werkomgeving geoptimaliseerd. Daarnaast heb je een snippet gemaakt voor sneller werken. Je beseft je dat je vs code kan personaliseren zoals je dat zelf wilt. Dit zorgt dat je steeds sneller kan programmeren.

In het begin zou ik zoveel mogelijk leren werken met de basis instellingen. Je kunt je backup en sync settings instellen zodat je extensies en instellingen van vs code gekoppeld worden aan je GitHub account. Dit zorgt dat je snel je werkomgeving kan herstellen.
