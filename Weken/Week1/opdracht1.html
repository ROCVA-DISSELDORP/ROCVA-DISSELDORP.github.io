Opdracht 1: VS Code Installeren

Beschrijving

In deze opdracht leer je Visual Studio Code (VS Code) installeren en de Live Server-extensie toevoegen om eenvoudig je websites te bekijken.

Stappen:

  1. Ga naar de officiÃ«le VS Code-website en download de editor voor jouw besturingssysteem.
  2. Installeer VS Code op je computer.
  3. Open VS Code, klik op de extensies-icoon (links in het menu), en zoek naar "Live Server".
  4. Klik op de knop "Installeren" bij de Live Server-extensie.
  5. Controleer of de installatie werkt door een HTML-bestand te openen en op "Go Live" te klikken rechtsonder in VS Code.

Resultaat:

Na het voltooien van deze opdracht kun je jouw HTML-bestanden live bekijken in je browser.
