Opdracht 2: Eerste Website Prototype

Beschrijving

In deze opdracht ga je jouw eerste website prototype genereren met behulp van een AI zoals ChatGPT of Claude AI.

Stappen:

  1. Open een AI-tool zoals ChatGPT of Claude AI.
  2. Stel een vraag zoals: "Maak een eenvoudige snake spelletje met HTML, Css en JavaScript."
  3. Kopieer de gegenereerde code en plak deze in nieuwe bestanden in VS Code (bijvoorbeeld index.html, styles.css en main.js).
  4. Gebruik de Live Server-extensie (zie Opdracht 1) om je website in de browser te bekijken.
  5. Experimenteer met de code: verander kleuren, voeg elementen toe, of verbeter de functionaliteit.

Resultaat:

Je hebt een werkend websiteprototype, zoals een snake-spel of een andere interactieve pagina.
