Openstaande Vacatures

Software Developer (PHP/Vue/Nuxt Focus)

Ben jij een enthousiaste Software Developer die graag wil groeien? Kom ons multidisciplinaire Agile team versterken! Je werkt mee aan de doorontwikkeling van onze innovatieve SaaS producten en leert werken met een moderne tech-stack.

Jouw Rol & Wat Je Gaat Leren:

  • Meewerken aan de ontwikkeling van backend services met Laravel (PHP).
  • Implementeren van user-facing features in de frontend met Vue.js of Nuxt.js.
  • Werken met PostgreSQL databases: data modelleren, queries schrijven en gebruik maken van de ORM (Eloquent).
  • Leren integreren met verschillende API's (voornamelijk RESTful).
  • Schrijven van unit tests (ervaring met TDD/PHPUnit/Jest is mooi meegenomen).
  • Actief gebruiken van Git voor versiebeheer via GitHub (denk aan branches, Pull Requests). Dit is essentieel voor teamwerk.
  • Kennismaken met en bijdragen aan onze deployment processen naar Azure (via CI/CD pipelines).
  • Rekening houden met security principes (input validatie, basisbegrip van JWT voor authenticatie).
  • Werken met UI componenten en styling via Tailwind CSS of Bootstrap (basiskennis HTML/CSS is nodig).
  • Meedenken over technische oplossingen en de architectuur van onze applicaties.
  • Helpen bij het opsporen en oplossen van bugs (troubleshooting).

Wat Wij Zoeken (Must Haves):

  • Aantoonbare programmeerervaring in PHP, bij voorkeur met kennis van Laravel (opgedaan via studie, projecten, stage of werk).
  • Goede basiskennis van JavaScript, HTML5, CSS3.
  • Ervaring met (of sterke interesse in) een modern JS framework zoals Vue.js of Nuxt.js.
  • Basiskennis van relationele databases (zoals PostgreSQL) en SQL.
  • Je kunt overweg met Git en platformen zoals GitHub.
  • Je begrijpt het basisidee achter API's (hoe applicaties met elkaar praten).
  • Basiskennis van hoe je een UI bouwt met frameworks zoals Tailwind/Bootstrap.
  • Je bent leergierig, communicatief en werkt graag in teamverband.

Bonuspunten (Nice to Haves):

  • Ervaring met cloud platforms, specifiek Azure.
  • Eerste ervaring met CI/CD tools (bv. GitHub Actions).
  • Kennis van containerization (Docker).
  • Bekendheid met Agile/Scrum werken.
  • Ervaring met Test-Driven Development (TDD).
  • Basiskennis van authenticatiemechanismen zoals JWT.
  • Affiniteit met DevOps-principes.

Wat Wij Bieden:

  • Een uitdagende functie waarin je veel kunt leren.
  • Goede begeleiding door ervaren collega's.
  • Een passend salaris en goede secundaire arbeidsvoorwaarden.
  • Volop mogelijkheden voor training en professionele ontwikkeling.
  • Een moderne werkomgeving met up-to-date tools.
  • Flexibiliteit in werktijden en mogelijkheid tot hybride werken.
