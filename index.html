Guide: Publish Your First Website

Set Up Git GitHub Pages Resources

1. Install and Set Up Git

Git is essential for version control and uploading your website to GitHub.

  1. Download and install Git from https://git-scm.com/downloads.
  2. After installation, open your terminal (Command Prompt, Git Bash, or VS Code terminal) and configure Git with your GitHub account: 
    
git config --global user.name "Your GitHub Username"
git config --global user.email "your-email@example.com"
  3. Verify your configuration with: 
    
git config --list

2. Create and Publish Your Website with GitHub Pages

Follow these steps to create and publish your website:

  1. Go to GitHub and create a new repository. Name it as your-username.github.io.
  2. Clone the repository to your local machine using: 
    
git clone https://github.com/your-username/your-username.github.io
  3. Add your website files (e.g., HTML, CSS, JS) to the cloned folder.
  4. Use the following commands to push your site to GitHub: 
    
git add .
git commit -m "Initial commit"
git push
  5. Open the repository settings on GitHub and enable GitHub Pages. Set the branch to main.
  6. Your website will be live at https://your-username.github.io.

3. Additional Resources
